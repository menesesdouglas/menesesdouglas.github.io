Python SQL Server tkinter

Sistema desktop em Python para gerenciar alertas agendados. Utiliza tkinter para a interface gráfica e SQL Server para armazenar os dados. Interface gráfica com tkinter
Banco SQL Server para persistência de dados
Notificações automáticas de alertas agendados
